gulp-quick-example
=====================

## Getting Started

```
npm install -g gulp
npm install
gulp
```

If you want to check htmlhint and csslint.

```
gulp lint
```

If you want to use livereload, please install [livereload chrome extension](https://chrome.google.com/webstore/detail/livereload/jnihajbhpnppcggbcgedagnkighmdlei).
